# Bike_Sales
🚴 Built an interactive Bike Sales Dashboard in Excel using 1000+ sales records to analyze sales trends, customer insights, payment methods, and business KPIs through data visualization and dashboard reporting.



🚴 Bike Sales Dashboard Analytics Project

This project is an interactive **Bike Sales Dashboard** developed in Microsoft Excel using a dataset of over **100,000 sales records**. The dashboard provides detailed business insights into bike sales performance, customer purchasing behavior, payment trends, and regional sales analysis.

The project involved complete data analysis steps including **data cleaning, preprocessing, KPI calculation, pivot table analysis, and dashboard visualization**. The dashboard was designed to help businesses monitor sales performance and make data-driven decisions efficiently.

🔹 Key Insights & Analysis:

* Total Revenue, Quantity Sold, Customers & Salesperson KPIs
* Month-wise and Year-wise Sales Trend Analysis
* Bike Category Performance Analysis
* Store Location & Regional Sales Comparison
* Payment Method Usage Analysis
* Customer Age & Gender-Based Insights
* Interactive Charts and Visual Reports for easy decision-making

🛠️ Tools & Technologies Used:

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* KPI Reporting
* Dashboard Design & Visualization
* Business Intelligence & Data Analytics

`Data Analysis` • `Excel Dashboarding` • `Business Intelligence` • `Data Visualization` • `Analytical Thinking` • `Reporting` • `Problem Solving` • `KPI Monitoring`
